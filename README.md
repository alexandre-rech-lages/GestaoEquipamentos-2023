# Gestão de Equipamentos

Gestão de Equipamentos é uma aplicação console construída durante a **9ª edição**, da [Academia do Programador](https://www.academiadoprogramador.net "Site da Academia do Programador").

A aplicação consiste no cadastro de equipamentos e controle de chamados para estes equipamentos.

## Funcionalidades

### Cadastro de Equipamentos  

	Requisito 1.1: Registro de Equipamentos  
	Requisito 1.2: Visualização de Equipamentos registrados
	Requisito 1.3: Edição de Equipamentos já registrados
	Requisito 1.4: Exclusão de Equipamentos já registrados

### Controle de Chamados  

	Requisito 2.1: Registro de Chamados
	Requisito 2.2: Visualização de Chamados registrados
	Requisito 2.3: Edição de Chamados já registrados
	Requisito 2.4: Exclusão de Chamados já registrados

# Tecnologias utilizadas

- .net core
- Linguagem C#
- Pilares da Orientação a Objetos 

# Como executar o projeto

### Pré-requisitos: dotnet 6.0

``` 
# clonar repositório
git clone https://github.com/alexandre-rech-lages/GestaoEquipamentos-2023.git

# entrar na pasta do projeto
cd GestaoEquipamentos.ConsoleApp

# compilar o projeto
dotnet build

# executar o projeto
dotnet run
```
